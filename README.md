# CroqueurDePomme
Réecriture du site croqueur de pomme de touraine en full stack
Front en Angular,
Back en Spring,
Base de donnée Mysql.

Le projet c'est transformé en la création d'un site de forum générique avec création de section, d'article, gestion d'utilisateur et présence de modération
